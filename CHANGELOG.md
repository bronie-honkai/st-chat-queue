# Changelog

All notable changes to this project will be documented in this file.

## [1.0.17] - 2025-12-27

### Added
- **Copy Button Feature**: Added a copy button to each queue item for easy text copying
  - Users can now quickly copy queue item text to clipboard
  - Visual feedback with toast notifications on success/failure
  - Button placed alongside other queue item actions (preview, edit, attach)

### Features
- Copy to clipboard functionality using modern Clipboard API
- Support for copying empty items (shows "(空文本)" placeholder)
- Event handling to prevent unintended interactions

---

## [1.0.16] - Previous Release

### Added
- Initial queue management system
- Drag and drop support for queue items
- Text editing functionality
- File attachment support
- Batch processing with AI integration
- Preview functionality for text and images
- Status tracking (pending, sending, done, error)

---

## Unreleased

### Planned Features
- Enhanced clipboard detection for legacy browser support
- Batch copy functionality for multiple items
